# Task2-Mobile

## About

This is a mobile testing application called "Diet_meal.apk". The testing is performed using WebdriverIo and Appium.

## Installation and Run

### Install package

Before running the script, make sure to install the required packages:

```bash
npm install
```
Run the script

```bash
npm run wdio
```
